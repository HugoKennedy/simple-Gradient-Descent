# Simple Gradient Descent

Ce code simule un neurone artificiel unique qui apprend par "essais et erreurs" :

Il fait une prédiction sur des données.

Il calcule son erreur par rapport à la vraie réponse.

Il corrige ses propres réglages (poids) pour ne plus refaire la même erreur.

En résumé, c'est un petit programme qui s'entraîne à classer des données (0 ou 1) en utilisant la bibliothèque NumPy.
